How does AutoSEO work?

AutoSEO turns a single URL into a prioritized, fixable report in one pass. It mirrors the four-stage pipeline from the AutoSEO/AutoGEO spec — crawl, diagnose, prioritize, then write the fix — so every issue arrives with a ready-to-apply correction rather than a vague warning.

  1. Fetch. The page is fetched server-side, following redirects and reading robots.txt — no browser CORS limits.
  2. Audit. Five auditors inspect on-page tags, technical health, structured data, social cards, and GEO signals.
  3. Prioritize. Findings are scored by severity and rolled up into a 0–100 grade with a per-category breakdown.
  4. Solve. Each fixable finding is turned into copy-paste-ready output — titles, meta, JSON-LD, TL;DRs, and Open Graph tags.

What is GEO and why does it matter?

GEO — Generative Engine Optimization — is optimizing for AI answer engines like ChatGPT, Perplexity, Claude, Gemini, and Google AI Overviews, where most sessions now end without a click to your site. Because roughly 44% of AI citations are lifted from the first 30% of a page, AutoSEO rewards pages that front-load a clear, self-contained answer, back claims with quotable statistics, and structure content as the questions people actually ask.

Do I need an API key?

No. AutoSEO is fully usable offline: without a key it returns deterministic, rule-based fixes. Set ANTHROPIC_API_KEY and Claude writes the actual corrected copy — better titles, descriptions, schema, and summaries — in a single structured call.
